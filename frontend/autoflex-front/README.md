# AutoflexFront

Este projeto foi gerado usando o [Angular CLI](https://github.com/angular/angular-cli) versão 21.0.5.

## Servidor de desenvolvimento

Para iniciar um servidor de desenvolvimento local:

Acesse a pasta:
```bash
cd /frontend/autoFlex-front
```
Execute:
```bash
ng serve
```

Assim que o servidor estiver em execução, abra seu navegador e acesse `http://localhost:4200/`. O aplicativo será recarregado automaticamente sempre que você modificar algum dos arquivos de origem.

## Building

Para construir o projeto, execute:

```bash
ng build
```
